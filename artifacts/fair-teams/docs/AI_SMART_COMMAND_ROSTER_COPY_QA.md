# AI Smart Command — Roster Copy Q&A

Adds Fair Teams knowledge for copying/duplicating rosters.

## User-facing answer target

For local/private rosters, explain the safe current path:

1. Open Roster Tools.
2. Open Local Backup.
3. Tap **Export this roster**.
4. Tap **Import one roster** and choose the exported file.
5. Fair Teams imports it as a separate new roster with a unique name; the original remains unchanged.

For shared rosters, explain **Make private copy** instead: it keeps the shared roster online and unchanged, creates a clean local copy, uses shared names and Club averages as starting skill, and resets photos/special abilities/private advanced traits.

## Why

Questions like “copy one roster to a new roster” previously routed to the generic/core answer because they mentioned “roster” but not “local/shared roster.” This patch adds a broader roster-copy topic and synonym routing for copy, duplicate, clone, export/import, new roster, and private copy wording.
